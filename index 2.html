OpenAI API Key Nasıl Alınır?

Uzantılarımızı kullanabilmek için bir OpenAI API key'e ihtiyacınız var. Eğer ChatGPT Plus kullanıcısıysanız, bu key'i zaten oluşturabilirsiniz.

1. OpenAI Key Sayfası'na gidin
2. "Create new secret key" butonuna tıklayın
3. Key'i kopyalayın ve uzantı popup'ına yapıştırın

Bu key sadece sizin tarayıcınızda saklanır. Bizimle veya kimseyle paylaşılmaz. Güvenle kullanabilirsiniz.
